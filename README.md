## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Swathi-ChandraSekar/portfolio-swathi.git
```

Navigate to project folder:

```
cd portfolio-swathi
```

Install dependencies:

```
npm install
```

Run the app:

```
npm run dev
```
